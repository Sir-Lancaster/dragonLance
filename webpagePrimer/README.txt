# How to Open index.html in Your Browser

## For Windows 10 and Windows 11 Users:
1. Open File Explorer.
2. Navigate to the directory where `index.html` is located.
3. Locate the `index.html` file.
4. Double-click on the `index.html` file.
5. Your default web browser will open and display the `index.html` file.

## For Mac Users:
1. Open Finder.
2. Navigate to the directory where `index.html` is located.
3. Locate the `index.html` file.
4. Double-click on the `index.html` file.
5. Your default web browser will open and display the `index.html` file.
